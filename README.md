# VOOMA Risk Intake Project

This project package was rebuilt from the last stable HTML version.

## Files
- index.html
- css/styles.css
- js/app.js
- js/config.js
- docs/notes.md

## Setup
Update `js/config.js` with your:
- Zapier webhook URL
- Calendly URL
- Canopy Connect URL
- Google Maps API key

## Deploy
For Netlify:
- Build command: leave blank
- Publish directory: .

## Notes
This package uses relative paths:
- css/styles.css
- js/config.js
- js/app.js

So `index.html`, `css`, and `js` must all stay at the same repo root level.
